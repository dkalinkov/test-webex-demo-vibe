Setup Instructions

  1. Create a Webex developer account at developer.webex.com
  2. Create a new integration with required scope: spark:people_read
  3. Get your access token from the developer portal
  4. Enter the token in the Authentication section above
  5. View your authenticated user information!

Features

  • Webex SDK integration via unpkg CDN
  • User authentication with access tokens
  • User information display
  • Token validation
  • Clean, simple interface
  • No build process required
